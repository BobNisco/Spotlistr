# Spotlistr
Spotlistr is the easiest way to create playlists for Spotify!

## Technologies Used
* [NodeJS](http://nodejs.org/)
* [AngularJS](https://angularjs.org/)
* [Spotify Web API](https://developer.spotify.com/web-api/)
